## 软件技术特点

### 开发语言
Python 3.9+

### 开发平台
- 操作系统：跨平台（Windows / macOS / Linux）
- 运行环境：Python 3.9 或更高版本

### 技术架构
采用前后端一体化的 Web 应用架构：
- **后端框架**：基于 Python 标准库构建的轻量级 HTTP 服务器
- **前端技术**：现代化响应式 Web 界面，支持桌面和移动设备访问
- **数据存储**：JSON 文件本地存储，支持多实体关系管理
- **权限控制**：基于角色的访问控制（RBAC）机制

### 主要功能模块

#### 1. 统一成果模型
- 支持六大成果类型：论文、专利、软件著作权、会议成果、项目材料、数据与代码
- 统一的 `ResearchOutput` 领域模型，覆盖所有成果类型的通用字段和专属字段
- 自动生成成果ID，按类型分配不同前缀（LW-论文、ZL-专利、RZ-软著等）

#### 2. 成员与项目管理
- 成员信息管理：ID、姓名、角色、联系方式、加入时间
- 项目信息管理：ID、名称、类型（基金/课题/协作）、负责人、起止时间
- 成果与成员、项目的多对多关联关系

#### 3. 权限与审核流
- 四级角色体系：PI、管理员、成员、只读用户
- 五状态审核流程：草稿 → 待审核 → 已退回/已通过 → 已归档
- 基于角色的操作权限控制（创建、编辑、提交、审核、删除）

#### 4. 外部数据抓取
- DOI 抓取：通过 CrossRef API 自动获取论文元数据
- PubMed 抓取：通过 NCBI PubMed API 获取生物医学文献信息
- 文档解析：支持上传 PDF、Word、TXT 文档自动识别标题、年份、摘要

#### 5. 统计与导出
- 成果汇总统计：按类型、年份、状态分类统计
- CSV 导出：导出成果清单为 CSV 格式
- Excel 高级导出：生成格式化的多工作表报告（汇总统计、成果清单、成员清单、项目清单）

#### 6. Web 管理界面
- 登录认证：基于 JWT 的会话管理
- 仪表盘：成果概览、快速统计、待审核提醒
- 成果管理：添加、编辑、删除、查看详情、状态流转
- 成员管理：添加、编辑、删除成员信息
- 项目管理：添加、编辑、删除项目信息
- 全面中文化：所有界面文本使用简体中文

### 数据安全特性
- 本地数据存储，数据不上传外部服务器
- 审计日志记录所有关键操作（创建、修改、删除、审核）
- 引用完整性检查，防止误删关联数据
- 备份与恢复机制（通过文件备份实现）

## 软件用途与应用范围

### 主要用途
为科研课题组提供科研成果的集中管理、审核、统计与导出功能，提升课题组科研资产管理效率。

### 适用对象
- 高校科研课题组
- 科研院所研究团队
- 企业研发部门

### 应用场景
- 课题组内部成果登记与审核
- 年度/项目周期成果汇总统计
- 科研考核材料准备与导出
- 成果资产追溯与管理

## 软件界面与操作方式

### 操作方式
- **Web 界面**：通过浏览器访问本地服务（http://127.0.0.1:8080）
- **命令行界面**：提供 CLI 工具进行批量操作和自动化管理

### 主要界面
1. **登录页**：用户身份认证入口
2. **仪表盘**：成果概览、统计图表、快速导航
3. **成果管理页**：成果列表、添加/编辑表单、详情页、抓取工具
4. **成员管理页**：成员列表、添加/编辑表单
5. **项目管理页**：项目列表、添加/编辑表单
6. **账号管理页**：修改密码、用户信息维护

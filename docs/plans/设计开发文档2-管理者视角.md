## 一、系统简介

科研成果管理软件是一款为科研课题组设计的成果信息管理工具。软件支持**论文、专利、软件著作权、会议成果、项目材料、数据代码**六大类型成果的统一管理，提供完整的审核流程和权限控制机制。

### 用户角色

| 角色 | 权限概述 |
|------|----------|
| **PI（课题组负责人）** | 最高权限，可审核成果、管理所有数据 |
| **管理员（Admin）** | 可管理成员、项目和成果，审核注册与注销申请，但**不能审核成果** |
| **成员（Member）** | 可添加和管理自己的成果、提交审核 |

> 成果审核权限仅归 PI；管理员如需审核，需由 PI 升级其角色为 PI。

## 二、管理者使用指南（软件全部内容）

管理者拥有比普通用户更高的权限，可以管理成员、项目，审核注册申请和注销申请，管理全部成果并进行统计导出。

### 1. 登录系统

管理员的登录方式与普通用户相同：

1. 访问系统地址
2. 输入管理员用户名和密码
3. 点击"登录"按钮

登录后即可看到管理员专属的功能入口。

### 2. 仪表盘

登录成功后进入软件仪表盘。相比普通用户，管理者的仪表盘额外汇总了待办审核提醒，便于快速进入管理工作。

- **成果统计概览**：
  - 总成果数量：系统中所有成果的总数
  - 按类型分类：论文、专利、软著、会议、项目材料、数据代码的数量
  - 按状态分类：草稿、待审核、已退回、已通过、已归档的数量
- **待办提醒（管理者专属）**：
  - 待审核账号：显示待处理的注册申请与注销申请数量，点击可直达"账号审核"页面
  - 待审核成果：PI 可看到需要审核的成果列表入口
- **快速操作入口**：添加成果、成员管理、项目管理、统计导出等常用功能的快捷入口
- **最近动态**：显示最近添加或更新的成果信息

### 3. 成员管理

- **添加成员**：进入"成员管理" → "添加成员"，填写：
  - 成员 ID（唯一标识，建议用工号或姓名拼音）
  - 姓名（必填）
  - 角色（PI / 管理员 / 成员 / 只读）
  - 邮箱、电话（可选）
- **编辑成员**：在成员列表点击"编辑"，修改后保存。
- **删除成员**：点击"删除"并确认。
  - ⚠️ 成员已关联成果（负责人/参与人）或项目（负责人）时，系统会阻止删除，需先解除关联。
- **查看详情**：点击成员姓名查看详细信息及其关联的成果列表。

### 4. 审核工作台

> 只有 PI 角色才能审核成果；管理员没有审核权限，需请 PI 审核或将角色升级为 PI。

审核工作台集中展示所有待审核成果，PI 在此完成成果审核。

PI 审核流程：

1. 进入"审核工作台"（或"成果管理" → "待审核成果"），页面顶部显示待审核数量徽标
2. 查看待审核成果列表
3. 点击成果标题查看详情
4. 做出决定：
   - **通过**：点击"审核通过"，状态变为"已通过"，所有用户可见
   - **退回**：点击"退回修改"，填写退回原因，成果退回给提交人

**成果状态标识**：🟡 草稿　🔵 待审核　🔴 已退回　🟢 已通过　⚫ 已归档

### 5. 账号审核

账号审核页面统一处理两类申请：**注册申请**与**注销申请**。入口为顶部菜单"账号管理" → "账号审核"，或在仪表盘点击"待审核账号"提醒。

#### 审核注册申请

当有新用户注册时，管理员需要审核其注册申请。

- **查看申请列表**：每条申请显示申请人的显示名称、申请用户名、注册时间。
- **处理申请**：
  - **通过申请**：点击"通过"，系统立即激活账号，用户可用注册的用户名密码登录。
  - **拒绝申请**：点击"拒绝"，系统删除该注册申请，用户需重新注册。

**审核建议**：核对显示名称是否为真实姓名；确认申请人是否为课题组成员；可疑申请建议线下确认身份后再审核。

#### 审核注销申请

当用户申请注销账号时，管理员需审核。

- **查看申请列表**：显示申请人姓名、用户名、申请注销时间。
- **处理申请**：
  - **批准注销**：点击"批准注销"并确认，账号被禁用无法再登录；该用户创建的成果**不会被删除**。
  - **拒绝注销**：点击"拒绝"，注销申请取消，用户可继续正常使用。

**审核建议**：确认申请人确实不再需要使用；检查是否有未完成成果；重要成员建议先线下沟通。

### 6. 项目管理

- **添加项目**：进入"项目管理" → "添加项目"，填写：
  - 项目 ID（如 `NSFC-2026-001`）
  - 项目名称（必填）
  - 项目类型（基金 / 课题 / 协作）
  - 负责人（必填，从成员列表选择）
  - 开始时间、结束时间、项目描述（可选）
- **编辑项目**：在项目列表点击"编辑"，修改后保存。
- **删除项目**：点击"删除"并确认。
  - ⚠️ 项目已关联成果时无法删除，需先解除关联或删除相关成果。
- **查看详情**：点击项目名称查看项目信息及该项目下的所有成果。

### 7. 成果管理

管理员可查看、编辑和删除系统中的**所有成果**（含其他成员创建的），并进行统计与导出。

- **查看所有成果**：进入"成果管理" → "成果列表"，可见所有状态（草稿、待审核、已退回、已通过、已归档）及所有成员创建的成果。
- **编辑他人成果**：在列表点击"编辑"，修改后保存。
  - 权限说明：可编辑任何**草稿状态**成果；已提交审核或已通过的成果需先退回为草稿。
- **删除成果**：在列表点击"删除"并确认。
  - ⚠️ 删除不可恢复，建议删除前与成果负责人确认。

#### 成果的完整录入内容（管理者亦按此录入/核对）

**成果类型**：论文、专利、软件著作权、会议成果、项目材料、数据与代码。

**基本信息（所有类型通用）**：

- 成果 ID（自动生成，可手动修改，如 `LW-2026-001`）
- 标题（必填）
- 年份（必填）
- 负责人（必填，可多选）
- 参与人、关联项目（可选）
- 摘要、关键词（逗号分隔）

**类型专属信息**：

| 类型 | 专属字段 |
|------|----------|
| 论文 | 论文类型（研究型/综述型/病例报告等）、期刊名称、DOI、PMID、影响因子、JCR 分区、投稿状态（写作中/已投稿/已接收/已发表）、第一作者、通讯作者 |
| 专利 | 专利号、申请号、发明人、申请日期、授权状态 |
| 软件著作权 | 登记号、软件全称、版本号、开发完成日期、首次发表日期 |

**外部数据抓取（论文）**：点击"从外部数据源抓取" → 选择"DOI"或"PubMed" → 输入 DOI（如 `10.1038/nature12345`）或 PubMed ID → 点击"抓取"，系统自动填充标题、作者、期刊、年份等，再核对补充必填字段。

#### 统计与导出

- **查看统计**：点击"统计"菜单，查看：
  - 成果类型分布、年份分布、状态分布
  - 成员贡献排行、项目产出统计
- **导出 Excel 报告**：点击"导出" → "Excel 报告"，生成含 4 个工作表的报告：
  - 汇总统计（类型/年份分布）
  - 成果清单（所有成果详情）
  - 成员清单、项目清单
  - 下载并保存文件

### 8. 账号设置

- **修改密码**：点击右上角用户名 → "账号设置"，输入当前密码，再输入新密码（≥8 位且包含字母和数字），确认后保存。
- **账号安全建议**：默认管理员账号首次登录后应立即修改密码；定期更换密码，妥善保管。

### 9. 退出登陆

点击右上角用户名 → "退出登录"，系统清除登录状态并跳转到登录页。

## 三、普通用户功能（简要说明）

普通用户（成员）功能是管理者能力的子集，主要包括：

- **注册账号**：登录页点击"注册"，填写显示名称、用户名（仅字母/数字/下划线）、密码（≥8 位且含字母和数字），提交后**等待管理员审核**（通常 1–2 个工作日），通过后方可登录。
- **登录系统**：用注册的用户名密码登录；未审核通过会提示"账号待审核"。
- **浏览仪表盘**：查看成果统计概览（总数、按类型/状态分类）、快速操作入口、最近动态。
- **添加成果**：选择成果类型 → 填写基本信息与类型专属信息 → 可用 DOI/PubMed 抓取辅助 → 以"草稿"状态保存，可随时继续编辑。
- **提交审核**：确认信息完整后对草稿点击"提交审核"，状态变为"待审核"，等待 PI 通过或退回（退回后可查看原因、修改再提交）。
- **查看成果**：可查看**自己创建的全部成果**及**其他成员已通过审核的成果**；支持按类型/状态/年份筛选和标题搜索；点击标题查看详情（含审核历史、关联项目、附件等）。
- **申请注销**：在"账号管理"申请注销账号，需管理员审核；审核通过前仍可正常使用，通过后账号被禁用，其创建的成果不会被删除（转交管理员）。
- **退出登录**：右上角用户名 → "退出登录"。

> 普通用户**不能**：审核成果、管理成员/项目、审核注册/注销申请、编辑或删除他人成果、查看统计与导出报告。

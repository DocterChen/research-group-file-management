# 科研成果管理 MVP 需求说明

## What

本 MVP 用于课题组内部以本地 CLI 形式管理科研成果与基础审核流，优先验证统一领域模型、角色权限和可追溯数据结构。

## 当前覆盖范围

- 成果类型：文章、专利、软件著作权、会议成果、项目材料、数据代码。
- 实体：成果、成员、项目、审核记录、审计日志。
- 操作：新增成员、新增项目、新增成果、提交审核、审核通过、查看列表、查看详情、汇总统计、CSV 导出。
- 存储：本地 JSON 多文件结构。

## 目标用户

- `pi`：查看全组成果、执行审核、查看统计与导出。
- `admin`：协助维护成果与审核。
- `member`：维护自己负责的成果并提交审核。
- `readonly`：只读查看。

## MVP 非目标

- 真实附件上传与下载
- 在线协作与登录系统
- Web UI
- 外部数据库
- PubMed / DOI / 专利数据库自动抓取

## 验收标准

- 可以通过 CLI 完成成员、项目和文章成果的完整录入。
- 普通成员只能提交自己负责的成果，不能审批。
- PI 可以审批已提交成果。
- 所有成果、审核记录和审计日志都可持久化到本地 JSON。
- 可以输出汇总统计和 CSV 文件。

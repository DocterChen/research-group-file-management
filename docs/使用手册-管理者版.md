## 一、系统简介

科研成果管理系统是一款为科研课题组设计的成果信息管理工具。系统支持**论文、专利、软件著作权、会议成果、项目材料和数据代码**六大类型成果的统一管理，提供完整的审核流程和权限控制机制。

### 用户角色

系统支持四种用户角色：

| 角色 | 权限说明 |
|------|----------|
| **课题组负责人（PI）** | 拥有最高权限，可以审核成果、管理所有数据 |
| **管理员（Admin）** | 可以管理成员、项目和成果，审核注册与注销申请，可以审核成果 |
| **成员（Member）** | 可以添加和管理自己的成果，提交审核 |
| **只读（Read-only）** | 只能查看已通过审核的成果 |

> **重要提示**：管理员和 PI 都拥有成果审核权限。

## 二、首次初始化

### 工作区设置

首次访问系统时，需要完成工作区初始化：

1. **设置组织信息**
   - **组织名称**：例如"马老师课题组"、"生物医学研究室"
   - **工作台副标题**：例如"成果管理与审核工作台"

2. **创建管理员账号**
   - **用户名**：建议使用工号或姓名拼音
   - **显示名称**：管理员真实姓名
   - **密码**：至少 8 位，包含字母和数字

3. **完成初始化**
   - 点击"创建管理员"按钮
   - 系统自动跳转到登录页面

> **提示**：管理员账号只用于工作区初始化和系统管理，建议不要与个人成员账号混用。

## 三、管理者使用指南

### 1. 登录系统

1. 访问系统地址（浏览器打开 Web 界面）
2. 输入管理员用户名和密码
3. 输入验证码（页面右侧显示的 4 位数字）
4. 点击"登录"按钮

登录成功后，系统会自动跳转到仪表盘页面。

---

### 2. 仪表盘

登录成功后进入系统仪表盘，这是管理者工作的起点。

#### 2.1 成果统计概览

实时展示系统中的成果数据汇总：

- **总成果数量**：当前账号可见的所有成果总数
- **按类型分类**：
  - 论文数量
  - 专利数量
  - 软件著作权数量
  - 会议成果数量
  - 项目材料数量
  - 数据代码数量
- **按状态分类**：
  - 草稿数量
  - 待审核数量
  - 已退回数量
  - 已通过数量
  - 已归档数量

#### 2.2 数据可视化

- **成果类型分布**：横向条形图展示各类型成果的数量对比
- **审核状态分布**：展示不同审核状态的成果数量
- **年度成果分布**：按发表/归档年份统计成果数量

#### 2.3 快速操作入口

常用功能的快捷按钮：

- 查看成果
- 导出 Excel
- 成员管理（管理员/PI）
- 项目管理（管理员/PI）
- 统计导出（管理员/PI）
- 账号审核（管理员/PI）

#### 2.4 最近成果

显示最近添加或更新的 5 条成果记录，方便快速追踪团队动态。

---

### 3. 成员管理

管理员可以完整管理课题组所有成员的账号信息。

#### 3.1 添加成员

1. 进入"成员管理"菜单
2. 点击"添加成员"按钮
3. 填写成员信息：
   - **成员编号**（必填）：唯一标识，建议使用工号或姓名拼音
   - **姓名**（必填）：成员真实姓名
   - **角色**（必填）：从以下选项中选择
     - PI（课题组负责人）
     - 管理员（Admin）
     - 成员（Member）
     - 只读（Read-only）
   - **邮箱**（可选）：用于接收通知
   - **备注**（可选）：补充说明
4. 点击"保存"完成添加

#### 3.2 编辑成员

1. 在成员列表中找到需要编辑的成员
2. 点击该成员行或成员编号进入详情页
3. 点击"编辑"按钮
4. 修改相应信息
5. 点击"保存"确认修改

#### 3.3 提升为管理员

在成员详情页，管理员和 PI 可以将普通成员提升为管理员：

1. 进入成员详情页
2. 点击"设为管理员"按钮
3. 确认提升操作

#### 3.4 删除成员

1. 在成员详情页点击"删除"按钮
2. 系统会弹出确认对话框
3. 确认删除

> **⚠️ 重要提示**：如果成员已关联成果（作为负责人或参与人）或项目（作为负责人），系统会阻止删除操作。需先解除关联或将成果/项目转交给其他成员。

---

### 4. 账号审核

账号审核页面统一处理两类申请：**注册申请**与**注销申请**。

**入口位置**：
- 顶部菜单：账号管理 → 账号审核
- 仪表盘：点击导航栏的"账号审核"（显示待审核数量徽标）

#### 4.1 审核注册申请

当有新用户注册时，管理员需要审核其注册申请。

**查看申请列表**

列表显示以下信息：
- 申请人的显示名称（真实姓名）
- 申请用户名（登录账号）
- 注册时间

**处理申请**

**方式一：通过申请**
1. 点击该申请行的"通过"按钮
2. 系统立即激活账号，并自动在成员列表中创建对应成员记录
3. 用户可以使用注册的用户名和密码登录系统

**方式二：拒绝申请**
1. 点击该申请行的"拒绝"按钮
2. 系统删除该注册申请
3. 用户需重新注册

**审核建议**：
- 核对显示名称是否为真实姓名
- 确认申请人是否为课题组成员
- 对可疑申请建议线下确认身份后再审核
- 审核处理时间建议控制在 1-2 个工作日内

#### 4.2 审核注销申请

当用户申请注销账号时，管理员需要审核。

**查看申请列表**

列表显示以下信息：
- 申请人姓名
- 申请人用户名
- 申请注销时间

**处理申请**

**方式一：批准注销**
1. 点击该申请行的"批准注销"按钮
2. 系统弹出二次确认对话框
3. 确认后，账号被删除，无法再登录
4. **重要**：该用户创建的成果**不会被删除**，会保留在系统中

**方式二：拒绝注销**
1. 点击该申请行的"拒绝"按钮
2. 注销申请取消
3. 用户可以继续正常使用系统

**审核建议**：
- 确认申请人确实不再需要使用系统
- 检查该用户是否有未完成的成果（草稿状态）
- 对重要成员建议先线下沟通确认
- 提醒用户注销后无法恢复账号

---

### 5. 审核工作台

审核工作台集中展示所有待审核成果，管理员和 PI 在此完成成果审核工作。

#### 5.1 审核流程

1. **进入审核工作台**
   - 方式一：点击导航栏"审核工作台"
   - 方式二：点击仪表盘的相关成果链接
   - 页面顶部会显示待审核数量徽标

2. **查看待审核列表**
   - 列表显示所有状态为"待审核"的成果
   - 包含：成果编号、标题、类型、状态

3. **快速审核**
   
   在列表页可以直接操作：
   
   - **通过**：点击"通过"按钮，成果状态变为"已通过"
   - **退回**：点击"退回"按钮，成果退回给提交人修改

4. **详细审核**
   
   点击成果标题进入详情页，仔细核对信息后：
   
   - **审核通过**：点击"审核通过"按钮
   - **退回修改**：点击"退回修改"按钮，可填写退回原因

5. **归档成果**
   
   对于已通过审核的成果，可以进行归档：
   
   - 在成果详情页点击"归档"按钮
   - 成果状态变为"已归档"

#### 5.2 成果状态标识

系统使用颜色标识区分成果状态：

| 状态 | 标识 | 说明 |
|------|------|------|
| 草稿 | 🟡 灰色 | 成果正在编辑中，未提交审核 |
| 待审核 | 🔵 黄色 | 成果已提交，等待审核 |
| 已退回 | 🔴 红色 | 退回修改，提交人需重新编辑 |
| 已通过 | 🟢 绿色 | 审核通过，对所有人可见 |
| 已归档 | ⚫ | 成果已归档存储 |

---

### 6. 项目管理

管理员可以管理课题组的所有项目信息。

#### 6.1 添加项目

1. 进入"项目管理"菜单
2. 点击"添加项目"按钮
3. 填写项目信息：
   - **项目编号**（必填）：唯一标识，如 `NSFC-2026-001`
   - **项目名称**（必填）：项目完整名称
   - **项目类型**（必填）：如"国家自然科学基金"、"省级课题"
   - **负责人**（必填）：可勾选成员或手动输入
   - **资助来源**（可选）：资助单位或部门
   - **开始年份**（可选）：项目起始年份
   - **结束年份**（可选）：项目结题年份
4. 点击"保存"完成添加

#### 6.2 编辑项目

1. 在项目列表中点击项目编号进入详情页
2. 点击"编辑"按钮
3. 修改相应信息
4. 点击"保存"确认修改

#### 6.3 删除项目

1. 在项目详情页点击"删除"按钮
2. 系统会弹出确认对话框
3. 确认删除

> **⚠️ 重要提示**：如果项目已关联成果，系统会阻止删除操作。需先解除关联或删除相关成果。

---

### 7. 成果管理

管理员可以查看、编辑和删除系统中的**所有成果**。

#### 7.1 查看所有成果

1. 进入"成果管理"菜单
2. 可以看到所有状态的成果
3. 支持筛选与搜索：
   - 按成果类型筛选（论文、专利、软著等）
   - 按成果状态筛选
   - 标题或编号关键词搜索

#### 7.2 分页浏览

- 成果列表默认每页显示 20 条记录
- 页面底部显示分页导航
- 显示当前页码和总页数
- 提供"上一页"和"下一页"按钮

#### 7.3 添加成果

有三种方式添加成果：

**方式一：手动录入**

1. 点击"添加成果"按钮
2. 选择成果类型
3. 填写基本信息和类型专属字段
4. 点击"保存草稿"或"提交审核"

**方式二：外部数据抓取**

1. 点击"添加成果"页面的"从 DOI / PubMed / 专利抓取"按钮
2. 选择抓取类型：
   - DOI / CrossRef
   - PubMed / PMID
   - 专利号 / 申请号
3. 输入检索值（如 DOI: `10.3390/biom12060824`）
4. 选择负责人
5. 点击"抓取并预填"
6. 系统自动填充字段后跳转到编辑页面
7. 补充必填信息后保存

**方式三：文档上传识别**

1. 在"外部数据抓取"页面上传论文 PDF、Word 或纯文本
2. 系统自动识别题名、年份、摘要、DOI / PMID
3. 预览识别结果
4. 确认后提交

> **提示**：外部数据抓取可大幅提高录入效率，建议优先使用。

#### 7.4 成果的完整录入内容

##### 基本信息（所有类型通用）

- **成果编号**：系统自动生成（可手动修改）
  - 论文：`LW-2026-001`
  - 专利：`ZL-2026-001`
  - 软著：`RZ-2026-001`
  - 会议：`HY-2026-001`
  - 项目：`XM-2026-001`
  - 数据：`SJ-2026-001`
- **成果标题**（必填）
- **成果类型**（必填）
- **年份**（可选）
- **负责人**（必填）：可勾选或手动输入
- **参与人**（可选）：可勾选或手动输入
- **关联项目**（可选）
- **关键词**（可选）：逗号分隔
- **摘要**（可选）
- **备注**（可选）

##### 类型专属信息

**论文类型**专属字段：
- 文章类型（必填）：如 Original Article、Review
- 期刊名称
- DOI
- PMID（PubMed ID）
- ISSN
- 投稿状态：如已发表、审稿中

**专利类型**专属字段：
- 专利号
- 申请号
- 专利标题
- 国家/地区
- 类型代码
- 专利状态：如申请中、已授权

**其他类型**：
软件著作权、会议成果、项目材料、数据与代码等类型的专属字段根据需要填写基本信息即可。

#### 7.5 编辑成果

1. 在成果列表或详情页点击"编辑"按钮
2. 修改成果信息
3. 点击"保存草稿"或"提交修改"

**权限说明**：
- 可以编辑任何**草稿状态**的成果
- 已提交审核或已通过的成果需先退回为草稿状态
- 建议编辑他人成果前与成果负责人沟通

#### 7.6 删除成果

1. 在成果详情页点击"删除"按钮
2. 系统弹出确认对话框
3. 确认删除

> **⚠️ 重要警告**：删除操作**不可恢复**，建议删除前与成果负责人确认，并做好数据备份。

#### 7.7 统计与导出

##### 查看统计信息

仪表盘自动展示各类统计数据：
- 成果类型分布（条形图）
- 年份分布（条形图）
- 状态分布（条形图）
- 最近成果（列表）

##### 导出 Excel 报告

1. 点击仪表盘的"导出 Excel"按钮
2. 系统自动生成完整报告
3. 下载文件（文件名：`research_outputs.xlsx`）

**报告包含的工作表**：
- 汇总统计
- 成果清单（所有成果详细信息）
- 成员清单
- 项目清单

> **用途**：导出的 Excel 报告可用于年度总结、项目结题、绩效考核等场景。

### 8. 账号设置

管理员可以查看自己的账号信息和申请注销。

#### 8.1 查看账号信息

1. 点击导航栏的"账号设置"
2. 查看以下信息：
   - 用户名
   - 显示名称
   - 角色
   - 成员编号
   - 账号状态
   - 创建时间

#### 8.2 申请注销账号

1. 在账号设置页面点击"申请注销账号"按钮
2. 确认注销申请
3. 等待管理员审批
4. 审批通过后账号被禁用
5. 自己创建的成果不会被删除

> **重要提示**：注销申请提交后，在管理员审批前仍可正常使用系统。

#### 8.3 账号安全建议

- **首次登录**：默认管理员账号首次登录后应立即修改密码（如需要）
- **定期更换**：建议每 3-6 个月更换一次密码
- **密码强度**：建议使用字母、数字、特殊符号组合
- **妥善保管**：不要与他人共享管理员密码
- **安全退出**：使用完毕及时退出登录

---

### 9. 退出登录

完成工作后，及时退出登录以保障账号安全。

**退出步骤**：
1. 点击导航栏的"退出登录"
2. 系统清除登录状态
3. 自动跳转到登录页面

> **安全提示**：在公共电脑或共享设备上使用后，务必退出登录。

## 四、普通用户功能简介

普通用户（成员）的功能是管理者能力的子集，主要包括：

### 4.1 注册账号

1. 在登录页点击"注册"按钮
2. 填写注册信息：
   - **显示名称**：真实姓名（必填）
   - **密码**：≥8 位且包含字母和数字（必填）
   - **用户名自动生成**：系统使用显示名称作为用户名
3. 提交注册申请
4. **等待管理员审核**（通常 1-2 个工作日）
5. 审核通过后方可登录

### 4.2 登录系统

- 使用注册的用户名和密码登录
- 输入验证码
- 未审核通过会提示"账号待审核"

### 4.3 浏览仪表盘

成员可以在仪表盘查看：
- 成果统计概览（仅自己可见的成果）
- 快速操作入口
- 最近动态

### 4.4 添加成果

1. 选择成果类型
2. 填写基本信息与类型专属信息
3. 可使用 DOI/PubMed/专利抓取功能辅助录入
4. 可上传文档自动识别
5. 以"草稿"状态保存
6. 可随时继续编辑草稿

### 4.5 提交审核

1. 确认成果信息完整准确
2. 在草稿成果详情页点击"提交审核"按钮
3. 成果状态变为"待审核"
4. 等待管理员或 PI 审核
5. 审核结果：
   - **通过**：状态变为"已通过"，所有人可见
   - **退回**：状态变为"已退回"，可查看退回原因，修改后重新提交

### 4.6 查看成果

成员可以查看：
- **自己创建的全部成果**（所有状态）
- **其他成员已通过审核的成果**

支持的功能：
- 按类型/状态筛选
- 标题关键词搜索
- 分页浏览
- 点击标题查看详情

### 4.7 申请注销

1. 在"账号设置"页面申请注销账号
2. 提交注销申请
3. 等待管理员审核
4. 审核通过前仍可正常使用
5. 审核通过后账号被禁用
6. 自己创建的成果不会被删除

### 4.8 退出登录

- 点击导航栏"退出登录"

### 4.9 权限限制

普通用户**不能**执行以下操作：
- ❌ 审核成果
- ❌ 管理成员（添加、编辑、删除成员）

- ❌ 审核注销申请

## 五、常见问题

### Q1：管理员可以审核成果吗？

**答**：可以。管理员（Admin）和 PI 都拥有成果审核权限。

### Q2：删除成员后，该成员的成果会被删除吗？

**答**：不会。删除成员账号不会删除其创建的成果，成果会保留在系统中。但如果成员仍关联着成果或项目，系统会阻止删除操作。

### Q3：批准用户注销后，其成果会被删除吗？

**答**：不会。用户注销后，账号被禁用无法登录，但其创建的成果会完整保留在系统中，由管理员管理。

### Q4：如何快速录入论文信息？

**答**：使用"外部数据抓取"功能，输入论文的 DOI 或 PubMed ID，系统会自动填充标题、作者、期刊等信息，大幅提高录入效率。也可以直接上传论文 PDF 文档，系统会自动识别。

### Q5：导出的 Excel 报告包含哪些内容？

**答**：包含汇总统计、成果清单（所有成果详情）、成员清单、项目清单。

### Q6：成果状态的含义是什么？

**答**：
- 🟡 **草稿**：正在编辑，未提交
- 🔵 **待审核**：已提交，等待审核
- 🔴 **已退回**：退回修改
- 🟢 **已通过**：审核通过
- ⚫ **已归档**：成果已归档

### Q7：如何修改组织名称？

**答**：组织名称在首次初始化时设置。如需修改，需要联系系统管理员直接修改数据目录下的 `workspace_settings.json` 文件。

### Q8：成果编号规则是什么？

**答**：系统自动按类型和年份生成编号：
- 论文：`LW-2026-001`
- 专利：`ZL-2026-001`
- 软著：`RZ-2026-001`
- 会议：`HY-2026-001`
- 项目：`XM-2026-001`
- 数据：`SJ-2026-001`

编号可以手动修改，但建议保持自动生成的规则以保持一致性。

### Q9：文档上传识别支持哪些格式？

**答**：支持 PDF、Word（.docx）、纯文本（.txt）、HTML、XML 格式。建议上传可复制文本的版本，扫描件图片识别效果可能不稳定。

### Q10：如何批量导入成果？

**答**：当前版本暂不支持批量导入。建议使用"外部数据抓取"或"文档上传识别"功能逐条快速录入。
